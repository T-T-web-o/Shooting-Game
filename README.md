# Shooting Game

## 概要
2Dシューティングゲームです。敵を倒すとスコアが増えボスを倒すとクリアになります。短時間で遊べることを意識して制作しました。

## 制作情報
- 制作期間: 2026年4月〜2026年5月
- 開発人数: 1人
- 担当範囲: 企画、実装、デバッグ、UI作成

## 使用技術
- C++
- Visual Studio
- Git
- DxLib

## 起動方法
1. GitHub右上の「Code」→「Download ZIP」を選択
2. zipを展開
3. 実行ファイルフォルダ内のTypingGame.exeを起動

## 操作方法
- 移動: W A S D
- 攻撃: Space

## 工夫した点
- TypingGame同様画面ごとにSceneを分割し、GameManagerでシーンの移行を管理しました。
また、unique_ptrを使用することで、メモリ管理を安全に行っています。
- 当たり判定では、画像のサイズを取得しオブジェクトのサイズに応じて判定を行いました。
なので画像変更後も当たり判定を調整しやすくなっています。
- プレイヤーがダメージを受けた際に無敵時間を設けることで理不尽な多重ヒットでのゲームオーバーを防ぎました。
